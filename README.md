# 🏋️ Gym Management System (Agile Scrum Project)
# 📌 Overview

This project demonstrates the implementation of Agile Scrum methodology using a Gym Management System as a real-world case study. The project was managed using Jira Scrum Board, simulating a complete sprint lifecycle including backlog creation, sprint planning, execution, and tracking.

The system is designed to manage:

Membership registration
Fitness class bookings
Attendance tracking
Payments
Equipment management
# 📌 Project Overview

The Gym Management System is a real-world inspired project built to demonstrate the Agile Scrum methodology using a Jira Scrum Board.

It simulates how a development team:

Plans tasks 📋
Executes sprints ⚙️
Tracks progress 📊
Delivers features incrementally 🚀
